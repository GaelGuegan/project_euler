# INSTALL INSTRUCTIONS

## DOWNLOAD

```shell
git clone https://github.com/GaelGuegan/project_euler.git
```

## DEPENDENCIES

```shell
 sudo apt-get install libgmp3-dev
 ```

## BUILDING

```shell
cd build
cmake ..
make
```

## TESTING

```shell
cd test
./run.sh
```

## INSTALL

```shell
sudo make install
```
